The MeetUp Moinator

This is a goofy little bookmarklet that turns profile pictures on Meetup.com 182% manlier.

Drag (or right-click and 'Add to Favorites') the MeetUp Moinator from that link right there into your bookmark bar. Then load any Meetup page with profile thumbnails (e.g. the October KWDM meetup page then click the link. Hairy!

Really bored? Check out the project on github at git@github.com:rmdrimmie/moinator.git
